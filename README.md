# Prework Study Guide Webpage

## Description

This Prework Study Guide was created to give students, in coding bootcamp, some basic knowledge and repitition before beginning the course.  The Webpage provides notes on HTML, CSS, Git and JavaScript.

## Installation

NA

## Usage

If you open the DevTools console, you can see the JavaScript running and it will randomly tell you what section to study first.  Each section has a small amount of notes on the given language.
